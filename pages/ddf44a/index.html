基因的本质(DNA)

# 1. DNA 是主要的遗传物质

# 1.1. 体内实验（肺炎双球菌体内转化实验）

# [1928 年格里菲斯]

  1. 实验材料：R 型肺炎双球菌（无多糖荚膜，无毒）、S 型肺炎双球菌（有多糖荚膜，有毒）、小鼠

  2. 实验原理：S 型肺炎双球菌使小鼠患败血症死亡。

  3. 实验过程及结果:

    • 过程：
      1. S 型菌可以使小鼠死亡；
      2. R 型菌不会使小鼠死亡；
      3. 加热杀死的 S 型菌不会使小鼠死亡；
      4. 加热杀死的 S 型菌和 R 型菌使小鼠死亡。
    • 结果分析：
      1. S 型活细菌有毒性
      2. R 型活细菌无毒性
      3. 加热杀死的 S 型细菌无毒性
      4. 加热杀死的 S 型细菌中，含有某种使 R 型活细菌转化为 S 型活细菌的转化因子
    • 实验结论：加热杀死的 S 型细菌中，含有某种使 R 型活细菌转化为 S 型活细菌的转化因子。

# 1.2. 体外实验

# [1994 艾弗里]

  1. 实验材料：R 型肺炎双球菌和 S 型肺炎双球菌

  2. 实验原理：R 型肺炎双球菌无多糖英膜，在培养基上形成的菌落表面粗糙；

    ​ S 型肺炎双球菌有多糖膜，在培养基上形成的菌落表面光滑。

  3. 设计思路：将 S 型细菌的提取物进行不同的处理，分别单独研究各种物质的作用

  1. 实验结论：只有用 DNA 酶处理的细胞提取物失去了转化活性，说明 DNA 是使 R 型细菌产生稳定遗传变化的物质。

# 1.3. 噬菌体实验

# [噬菌体侵染细菌的实验]

  1. 科学家：赫尔希、蔡斯（1952 年）

  2. 实验材料：T~2~噬菌体（一种专门寄生在大肠杆菌体内的病毒，由蛋白质和 DNA 成分构成）、大肠杆菌。

  3. 实验方法：同位素标记法

  4. 实验原理：T~2~噬菌体侵染大肠杆菌后，会在自身遗传物质的作用下，利用大肠杆菌体内的物质来合成自身的组成成分，进行大量繁殖。

  5. 设计思路：放射性同位素标记法，直接单独观察各自的作用。

  • 蛋白质：C、H、O、N、S → ^35^S 标记
  • DNA：C、H、O、N、P → ^32^P 标记

  3. 实验结论：DNA 是遗传物质。

# 1.4. 总结

# [DNA 是主要遗传物质]

大多数生物的遗传物质是 DNA，少数生物的遗传物质是 RNA。

因此DNA 是主要的遗传物质。

真核生物 原核生物 病毒
核酸 DNA 和 RNA DNA 和 RNA DNA 或 RNA
遗传物质 DNA DNA DNA 或 RNA

# [RNA 病毒]

烟草花叶病毒、流感病毒、禽流感病毒、HIV 病毒、冠状病毒 (SARS, MERS, COVID-19) 、甲肝病毒 HAV、丙肝病毒 HCV

# 2. DNA 的结构和复制

# 2.1. DNA 的结构

  1. 元素组成：C、H、O、N、P。

  2. 基本组成单位：脱氧核苷酸

    1. 物质组成：磷酸、脱氧核糖、含氮碱基 （腺嘌呤 A、鸟 嘌呤 G、胞嘧啶 C、胸腺嘧啶 T）

    2. 结构简式：

    3. 空间结构

    4. DNA 分子是由两条链组成的，这两条链按反向平行方式盘旋成双螺旋结构；

    5. 脱氧核糖和磷酸交替连接，排列在外侧，构成基本骨架，碱基排列在内侧；

    6. 两条链上的碱基按照碱基互补配对原则（A 与 T、C 与 G）通过氢键连接成碱基对。

# [DNA 分子复制小结]

定义 指以亲代 DNA 为模板合成子代 DNA 的过程。
场所 1. 真核细胞：细胞核（随着染色体的复制而完成）、线粒体、叶绿体；2. 原核细胞：拟核、细胞质（质粒）
时期 真核细胞核 DNA 的复制：有丝分裂的间期（S 期）和减数第一次分裂前的间期（S 期）
结果 1 个 DNA 分子一 2 个完全相同的 DNA 分子
模板链去向 分别进入 2 个子代 DNA 分子中
分别进入 2 个子代 DNA 分子中 碱基互补配对原则（A-T、T-A、 C-G、 G-C）
条件 1. 模板：亲代 DNA 的 2 条链；2. 原料：4 种游离的脱氧核苷酸；3. 引物：二小段 RNA；4. 酶：解旋酶 DNA 聚合酶；5. 能量：由 ATP 水解提供
特点 [多起点复制] [双向复制] [边解旋边复制] [半保留复制]
精准复制的保证 1. DNA 分子的双螺旋结构为复制提供了精准的模板；2. 通过碱基互补配对保证了复制的准确性
意义 DNA 分子通过复制，将遗传信息从亲代传给子代，从而保持了遗传信息的连续性。

# 3. DNA 的计算

徦设将 1 个全部被"N 标记的 DNA 分子转移到含”N 的培养基中培养 n 代，则：（该 DNA 分子中含有乌嘌呤 m 个）

  1. 子代 DNA 分子总数是 2^n^ 个
  2. 含 ^15^N 的 DNA 分子数是 2 个
  3. 只含 ^15^N 的 DNA 分子数是 0 个
  4. 含 ^15^N 的脱氧核苷酸链有 2 条
  5. 经过 n 次复制，共需消耗游离的鸟嘌呤 (2^n^ - 1)m 个
  6. 第 n 次复制时，共需消耗游离的乌嘌呤 (2^n-1^)m 个
